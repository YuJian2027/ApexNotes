# 上岸笔记 · 新电脑上手指南（2 分钟）

欢迎使用「上岸笔记」——国考行测错题追踪 Agent 技能。本包已为你准备好**空白起步**版本：
没有预置任何错题，所有错题由你自己的备考逐步录入积累。

## 一、环境准备（一次性）
- 安装 **Node.js 18+**：https://nodejs.org （装 LTS 即可）
- 安装 **Python 3**（可选，仅 Excel 导出需要）：https://python.org

## 二、安装（一次性）
在本文件夹内打开终端，执行：
```bash
bash install.sh          # macOS / Linux / Git Bash
# 或 Windows PowerShell：  bash install.sh
```
脚本会自动：检测 Node/Python → 初始化空错题库 → 解析 472 节点知识框架。
数据默认存在 `~/.apexnotes/data/`（Windows 在 `C:\Users\你\.apexnotes\data\`）。

## 三、接入 Agent
把本文件夹作为 Skill 加载（不同平台见 `PLATFORM_GUIDE.md`）：
- WorkBuddy / CodeBuddy：把本目录放进技能目录，对话中引用 `SKILL.md`
- 飞书等：参考 `assets/workspace-example.yaml` 配置定时任务

## 四、开始用（每天）
1. 截图错题或文字快捷录入，例如发一句：
   `判断推理-图形推理-粗心-待二刷` （换行后可附题干文字）
2. Agent 帮你定位知识点、生成确认卡片，你核对「归类 / 错因」后回复「对」入库
3. 每晚自动生成当日总结；复习按**艾宾浩斯遗忘曲线**排期（1→2→4→7→15→30→60→120 天），
   到期时 Agent 推送旧题让你自评「记得 / 不记得」

## 五、查看成果
- 知识树网页：`python scripts/build_unified_html.py` → 打开生成的 HTML
- Excel 导出：`pip install openpyxl && node scripts/export_xlsx.js`
- 完整说明：读 `README.md`（用户操作指南）与 `SKILL.md`（技能定义）

> 提示：你想把数据存到别的盘，安装前先设环境变量
> `export APEXNOTES_DATA_DIR=/你的/路径/data`（Windows 用 `set`）。
